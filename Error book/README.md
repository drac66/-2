# Error Book（多端错题本）

技术路线（按你新要求）：
- `apps/mobile_flutter`：手机端 Flutter（Android / iOS）
- `apps/desktop_java`：电脑端 Java（Swing）
- `packages/shared_schema`：跨端共享数据结构说明（JSON Schema）

## 功能目标
1. 错题添加
2. 错题查询（分类、关键词）
3. 错题复习（随机抽题）
4. 错题编辑 / 删除

## 端侧定位
- 手机端：快速录入 + 随手复习
- 电脑端：高密度管理（筛选 / 列表 / 详情）
